Introducing: Advanced Circle Path
It is also possible to create circular paths in Diesel Studio.

How?
1. Create a new scene and Select a sound source,
2. In [Set Path Type] select [Circle]
3. Open the [Circle Path] Panel
4. Set [Circle Radius]
5. Set [Circle Rounds/min]
6. Set [Circle Center] (X,Y,Z)
7. Play

You will now see (and hear) the sound source move in a circle in accordance with the stated values.
You can make a sound source rotate around itself. This function is combined with cone.

How?
1. Select a sound source, choose a circle path, create a circle path (follow the instructions in Circle Path)
2. Set an [object orientation] in the [Circle Path] panel
3. Go to the [Parameter] panel
4. Set Cone angles Inside (e.g. 90)
5. Set Cone angles Outside (e.g.120)
6. Drag Outside volume slide (e.g. -8)
7. Play

You will now be able to hear clearly how the sound not only circles around you but also alters in level in accordance with the selected cone values.
You can turn the circle in three elevations.

To follow the changes select (Update All views) and select [Play]
1. Go to [Circle Path] panel
2. Drag slider in [Circle Property]:
3. Change [Circle tilt] from 0 degrees to 45 degrees

You will now see [Back view (XZ elevation)], how the circle path changes from a linear movement where Z=O, to a linear movement where X=Z (which is the same as a displacement of the YX elevation by 45 degrees in the XZ elevation).

Change [Circle Elevation] from 0 degrees to 45 degrees, and set [Circle Tilt] to 0 degrees.You will now see [Side view (YZ elevation)], how the circle path changes from a linear movement where Z=O to a linear movement where Y=Z.
