object properties - line path

 

Line Path: to create and set a line path. 

In the line path edit control you have the option to: 

  • Create a new point with the "New Point" button. 
  • Delete a point with the "Delete Point" button. 
  • Distribute time with the "Distribute Time" button. 
  • Set position. 
  • Set orientation. 
  • Set interpolation type. 
  • Set line segment type. 

Create new point 
When the "New Point" button is pressed a dialogue appears where you can set the starting position and orientation. All values are default 0. If this is not the last point chosen in the list box the new point will be inserted between the selected point and the point following. If it is the last point which is selected a dialogue box will display "New end point". 

All points are represented with the point time in the list box. 

Delete point
A point is deleted with the "Delete Point" button. 

Distribute time 
The "Distribute Time" button is used to distribute time over a number of points. This distribution can be made either as "Constant velocity" or by using "Equal amount of time between the points" (regardless of the distance between the points). 

Position 
The position can be changed in real time by tracking the slider or by typing a new value in the edit box. The position is in meters and the range is set to be from -50 to 50 meters 

Orientation 
The orientation is in degrees and the range is set to be from 0 to 360 degrees. 

Interpolation 
It is possible to select between two types of orientation interpolation which can be used, when creating a path: 

  • Euler - Counter clockwise Rotation 
  • Quaternion - shortest X,Y,Z rotation 

Line Segment Type 
A line path can be created with different line types by selecting one of the line types in the "Line Segment Type" drop down list. The line type options are: 

  • Straight line. 
  • Left half circle in the XY elevation. 
  • Right half circle in the XY elevation. 
  • Left half circle in the XZ elevation. 
  • Right half circle in the XZ elevation. 
  • Left quarter circle in the XY elevation. 
  • Right quarter circle in the XY elevation. 
  • Left quarter circle in the XZ elevation. 
  • Right quarter circle in the XZ elevation.
