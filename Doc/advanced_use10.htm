Introducing: Interpolation

How?
1. Load DS_Tut6.dss
2. Select Play

You will now see the listener move around in a rather strange way. You can, by using interpolation, change the direction of the Listener to make him move forward through the path.

How?
1. Choose Interpolation/Quaternion
2. Select the first Time-point
3. Drag the slider in Orientation/AZ to 270.00
4. Select the second Time-point
5. Drag the slider in Orientation/AZ to 270.00
6. Select the second Time-point
7. Drag the slider in Orientation/AZ to 180.00
8. Continue with all the Time-points using Play as preview.
9. Select PlayWhen you have set the orientation/interpolation, the listener will move around in a quite natural way.

If you have any problems in creating the scene you can load DS_Tut7.dss to take a look at how the finished path can be.
