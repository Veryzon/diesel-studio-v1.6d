object properties - circle path

 

Circle Path: to create and set a circle path. 

In the circle path edits control you have the option to: 

  • Set circle property. 
  • Set object orientation. 
  • Set circle center. 

Circle Property 
The circle property contains the following parameters: 

  • Circle Tilt 
    The circle tilt is used to tilt the circle path x degrees between -180 and 180 degrees. Default is 0 degrees. 
  • Circle Elevation 
    The circle elevation is used to elevate the circle x degrees between 0 - and 180 degrees. Default is 0 degrees.
  • Circle Radius 
    The radius is used to set the radius of the circle between 0 and 100 meters. Default is 0 meters. 
  • Horizontal Circles 
    Sets the number of circle rounds per minute for the object. The range is -100 to 100 rounds/min. Default is 0 rounds/min. 
  • Offset 
    The azimuth offset is in degrees between 0 and 360 degrees and default is 0 degrees. 

Object Orientation 
The object orientation contains the following parameters: 

  • Object Orientation (Azimuth): 
    The rotation of the object in rounds per minute around the object's own z axis. The range is from -120 to 120 rounds/min and default is 0 rounds/min.
  • Object Orientation (Elevation): 
    The rotation of the object in rounds per minute around the objects own x axis. The range is from -120 to 120 rounds/min and default 0 rounds/min.
  • Object Orientation (Tilt): 
    The rotation of the object in rounds per minute around the object's own y axis. The range is from -120 to 120 rounds/min and default is 0 rounds/min. 

Circle Center 
The circle centre contains the x, y and z parameters for the positioning of the centre of the circle.

Reset All 
The "Reset All" button resets all parameters to the default settings. 

Note:
The circle path overrides the line path!
