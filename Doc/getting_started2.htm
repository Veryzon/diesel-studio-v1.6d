getting started - tutorial 2

 

Introducing: Movement, Circle, Zoom, Position
Get the Helicopter to fly around the listener in a circle of 10 m, at 5 rounds/min

How?
1. Select 'Helicopter' in the Object Edit box
2. Select Circle Path
3. Drag Slider Set [Circle Radius] to 10([Circle Radius] can be set in two ways: you can regulate by using the slide bar, or enter the required radius in the text box)
4. Drag Slider Set [Circle Rounds/min] to -5([Circle Rounds/min] can be set in two ways: you can regulate by using the slide bar or enter the required radius in the text box)
5. Select [Play]

You will now be able to hear that the helicopter has moved further away from you and that the sound has changed. You will, however, also experience that you can no longer see the sound source.

How?
Change the Zoom resolution, and change the Circle Center
1. Go to [Zoom]
2. Select 50%
3 Go to Circle Center
4. Type 8 in X
5. Type 8 in Y
6. Select Play

You have now changed your zoom resolution and center and you will now be able to hear the helicopter in front of you, and to the right of you.
