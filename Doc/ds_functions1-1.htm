editor window - object editor

 

The object editor is used to create new objects and delete objects in the acoustic environment. All the created objects are listed in a drop down list. The object selected in the drop down list is the object activated for the edit control. 

In the object editor the path type for each object is selected. Two path types are available; Line Path and Circle Path. Editing of the paths is carried out in the edit control. 

NOTE: A high number sound sources can be activated in Diesel Studio. Note that a large number of sound sources may lead to an insufficient update of the Graphical Presentation of the sound sources. In this case please minimize the program window before listening. 

Object edit 
The name of the selected object is accentuated in the text box.

Set Path Type 
In Diesel Studio you can choose between two path types 

  • Line Path - The choice of line path activates the tab under Edit control 
  • Circle Path - The choice of circle path activates the tab under Edit control 

New Source
In New Source you will be able to create a new sound source with start position and orientation. A wav file is associated to each sound source. A sound source is always set to a default wav file. It is possible to change the wav file by clicking the "Select file" button. Choose a sound file by browsing in the library in which the required sound file is placed. It is not possible to change the wav file after the sound source has been created.

A graphical 3D model (in 3ds format) can be associated with each sound source. If the size and/or centering of the model isn't suitable, then the auto scale and/or auto center options can be selected. If no 3D model is selected, then the default model (simple box) will be used.

The name of the object can be changed by typing the name in the "Object Name" edit box and by clicking the "OK" button. The sound source is created with the selected settings.

Delete Object 
Delete Object to delete a sound source. When the "Delete Object" button is pressed a dialogue appears to ensure that the selected object is, in fact, to be deleted.

Head Tracker 
Pressing the "Head Tracker" button will bring up a dialogue that allows to assign a head tracker sensor to the current object. The data from the head tracker sensor will control the position and orientation of the object. Additional controls are:

  • Object Pose
    • Absolute - Sensor data is used directly
    • Relative - Sensor data is added to the Line Path or Circle Path of the object
  • Calibrate Position - Resets the position of the selected sensor to 0,0,0 
  • Calibrate Orientation - Resets the orientation of the selected sensor to 0,0,0 
Tracker sensors belong to tracker devices (one tracker device can have several sensors), and each device can be controlled by the following buttons:
  • Start Device - The device starts receiving position and orientation of each of its sensors 
  • Stop Device - The device stops receiving 
  • Device Config - Configuration of the devices, with the following controls:
    • Add - Add a new instance of a device. The devices are made as plug-in DLLs, and are located under the \TrackerDlls\ directory. Contact tech@am3d.com for information about additional tracker devices.
    • Remove - Removes the selected device instance.
    • The plug-in module specific configuration can be adjusted in the "Device Configuration" group box.
