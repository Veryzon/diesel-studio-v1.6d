functions - toolbar

 

There are eight tools available in the toolbar window: 

  • Select object 
    The select object tool is used to select an object for editing. 
  • Move object 
    The move object tool is used to move the position of the object in the environment. 
  • Rotate view
    The rotate view tool is used to rotate the view.
  • Move view 
    The move view pointer is used to move the centre of the view.
  • Zoom in 
    Increases the zoom level (all views). 
  • Zoom out 
    Decreases the zoom level (all views). 
  • Play
    Starts the playback.
  • Stop 
    Stops the playback.

Note:
Changes made in the views using tool 3 and 4 can be reset by right clicking the mouse in the active view (resets the active view).
