advanced use - tutorial 9

 

Introducing; Advanced Movement/Velocity, Orientation
You have now created a path but before you can hear how it sounds you must of course state the speed at which you would like the object to move.

How?
1. Select all time points in Line Path
2. Select Distribute Time
3. Type your decided time (time is measured in minutes (00-), seconds (00-59) and frames (00-24))
4. Select equal amount of time between points
5. Select OK

You will now be able to see that the time-point has time values and that each point is separated by precisely the same time. If you had selected Constant Velocity you would have got a constant speed.

Now listen to your scene

The function of Play in Diesel Studio Double: When you select Play, the WAV file (or files) start playing with an automatic loop function and the scene is activated with the objects moving from start time-point to end time-point. When a sound source, for example, reaches the end time-point, you will hear the sound loop in that point until you choose stop.
As default, all sound sources in Diesel Studio are multidirectional. If you set the orientation on sound sources without stating the cone, you will get only a visual - non-audible alteration of your scene. See sections 3-6. This would, though, make sense if you had, instead, constructed a path with the listener.

Earlier, you constructed a [Line Path] with a sound source. Now you can try to construct a scene with the listener.

How?
1. Select Receiver, Create New Point, Distribute Time
2. Choose the second time-point
3. Drag the slider in Orientation/AZ to any different location (e.g. 180)
4. Select playYou will now see the head turn, shifting location at the same time. Orientation on receiver is very useful when you want to move around naturally in a sound environment.
The direction of turn is determined by using Interpolation.
