advanced use - tutorial 7

 

In the following tutorials you can learn more about how to use Diesel Studio.

Create your own scene In Diesel Studio you can construct your very own sound scenes. Read on, to learn more about how you make scenes and about the more advanced uses of Diesel Studio.

Introducing: WAV-file
This demo version of Diesel Studio is optimized for 44k1Hz and 22k05Hz playback. It is therefore recommended that a single channel, mono, 16 bit, 44k1Hz / 22k05Hz is used in this demo.

The use of other formats will cause deterioration in the sound quality, perhaps resulting in a program breakdown. The same sound file can be used for different sound sources in a scene.

How?
1. Go to New Source
2. Chose Select file
3. Browse for destination of audio file
4. Choose Open file
5. Type a name in Object Name
6. Choose OK

Now you have inserted your own sound file into a scene. The default position for this new sound file is origo (0,0,0). You can now put more sounds in or make a track/movement for the selected sound file.

In Diesel Studio we recommend to use a limited number of sound sources (up to 32), but be aware that, perceptually, we are only capable of distinguishing 6-8 sound sources.

There are two basic methods of making paths; lineal (Tutorial 8) or circular (Tutorial 11). In the real world, sound sources can either be stationary or mobile, in exactly the same way that the listener can stand still and listen, or move around in an environment. These possibilities also exist in Diesel Studio. Knowledge and experience in the area of how these possibilities can be utilized is very relevant to how "real" or "good" the sound scene is.
