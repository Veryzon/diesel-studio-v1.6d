getting started - tutorial 1

 

Introducing: Sound Source, Receiver (listener)
The listener is graphically represented by a head, and all sound sources by a box. In the Top View corner of the screen you can see a purple box and a head. You can move the sound source or the listener by first marking the object with the pointer "select object", and then choosing the pointer "move object", after which the object can be placed in the required position.

A start scene is installed in Diesel Studio. The scenario consists of a single sound, a helicopter and a listener.


How?
1. Go to: Settings/Playback
2. Select: Playback Settings (Headphones recommended)
3. Select: OK
4. Go to: File/Open
5. Choose: sample scene: Tutorial1.dss
6. Select: Play
