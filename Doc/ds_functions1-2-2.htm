object properties - parameter

 

The Parameter control is divided in five parameter types: 

Gain
The gain factor is in dB and is the overall volume for a sound source. The range of the gain factor is from -100 to 10 dB, default is 0 dB. 

Radiation 
The Source Radiation parameters are split into three parameters;

  • inside cone angle 
  • outside cone angle 
  • outside cone volume. 

The inside and outside cone angles are in degrees and the ranges of the angles are from 0 to 360 degrees and default is 360 degrees. The outside cone volume is in dB and the range is -100 to 0 dB and default is 0 dB.  

Distance 
The distance parameter types are split into three parameters; 

  • minimum distance 
  • maximum distance 
  • roll-off factor. 

The minimum and maximum distances are in meters and the ranges are from 0 to 100 meters. The minimum distance is default 1 meter and the maximum distance is default 100 meters. The range of the roll-off factor is 0 to10 and default is 1.  

Doppler 
The range of the Doppler factor is from 0 to 10 and default is 1.

Pitch 
The range of the Pitch factor is from 0.1 to 4.5 and default is 1. 

Reset All 
The "Reset All" button resets all parameters to the default values. Parameter: to set advanced parameters for the sound sources.
