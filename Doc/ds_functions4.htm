File

Here it is possible to create a new document, load a document, and save the document from the menu. The file format used has the extension *.dss and is stored in a binary windows format.

Settings

Contains the playback settings 

Playback Settings
The "Playback Control" dialogue contains the playback settings where the system can be set to run with loudspeakers or headphones. 

Headphones are set as the default selection. In the "Headphone types" list, different types of headphones can be selected to improve sound quality. 

If loudspeakers are selected you have the option to set the "Loudspeaker Angle". Selecting an angle that matches the listening situation is essential for the overall 3D effect. By typing the two distance parameters in the "Angle Calculator" the exact angle can be calculated.


Play Back Device 

Headphones 
When headphone playback is selected a number of settings are available, enabling more compatibility between the headphone set and the binaural synthesis engine. This is important both for positional performance and for achieving natural sound reproduction. Of course the nature of the input signal (selected wav file) also has a major influence on this. Headphone adaptation however, offers the possibility of making all headphones sound similar. 

If none of the options match the headphones, try each of the settings one by one until the 3D positional performance matches the best shown in the render view. 

Headphones Type

  • Neutral 
    No headphone adaptation is made. 
  • Free Field Calibrated 
    Use this setting when listening through free-field calibrated headphones.
    Some studio reference headphones are free-field calibrated, so check the manual. However, the standard used in this engine may deviate from the factory standards and therefore the optimum setting might be "Diffuse Field Calibrated". Most cheap headphones can, with advantage, be used with this setting as active. 
  • Diffuse Field Calibrated 
    Use this setting when listening through a diffuse-field calibrated headphone set.
    Some studio reference headphones are diffuse-field calibrated, so check the manual. Some headphones can also, with advantage, be used with this setting as active. 
  • Beyerdynamic DT 990 
    A setting made for listening through Beyerdynamic DT 990 headphones.

Loudspeakers 
When loudspeaker playback is selected a cross-talk cancellation algorithm is used in the post-processing. The algorithm assumes a symmetrical listening situation where the listener must be positioned in the middle between two loudspeakers. The purpose of the algorithm is to eliminate sound from the right loudspeaker at the left ear and vice versa. 

The so called sweet spot, which is the physical area where the 3D effect is optimal, is rather small and dependent on the distance to loudspeakers. Larger distances reduce the sweet spot's sensitivity. 

The loudspeakers should be placed at the same height as the ears of the listener. If this is not possible it is recommended to place the loudspeakers lower and not higher. If the system has a sub woofer, it may be necessary to switch this off in order to achieve a satisfactory 3D performance. 

Place the loudspeakers symmetrically and try to avoid placing them next to side walls. They can however be placed next to a monitor without influencing the 3D effect. If one speaker has to be placed next to a side wall, it is recommended that the same is done with the other, in order to avoid a deterioration of the 3D effect. 

  • Loudspeaker Angle 
    The angle spanned by the two loudspeakers, seen from the listener's position. 
  • Distance 1 
    The distance between the loudspeakers 
  • Distance 2 
    The angular distance from the loudspeakers to the listening position.
  • Angle 
    Shows the calculated angle. 
  • Calculate 
    Used to calculate the angle to be set in.


View
In this menu it is possible to enable or disable any of the four view windows, as well as the docking tool windows and toolbar.
Additionally it is possible to save the current window configuration (in an ini file), so that the currently open windows will be open next time the application is used, also preserving their docking state (unless the screen resolution changes).


Window
Here it is possible to arrange the windows and to select a window to be active.
