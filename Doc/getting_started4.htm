getting started - tutorial 4

 

Introducing: Distance, Roll-off

How?
1. Choose sample scene Tutotial3.dss
2. Select play
3. Select 'Footsteps' in the Object Edit box
4. Go to Parameter
5. Drag Slider Minimum slowly to 6

You will now be able to hear that the sound level has increased. This is because the 0 dB distance of the sound source has moved from 1 m to 6 m. Distance attenuation will, in other words, first begin at a distance of 6 m instead of 1 m.

How?
1. Choose Reset All in Parameter
2. Choose Line Path
3. Select New Point
4. Set (X,Y,Z) to (10.00; 8.00 ;0.00)
5. Distribute Time to 10:00
6. PlayNow you will hear a person come from front left, move to the right and then behind you.

You will be able to change the attenuation through the roll-off factor

How?
1. Choose Parameter
2. Set Roll Off to 3
3. Select Play

As the distance to the person increases the sound level of the footsteps decreases even faster. A roll-off factor higher than 1 increases the attenuation effect - a roll-off factor under 1 decreases the effect.
