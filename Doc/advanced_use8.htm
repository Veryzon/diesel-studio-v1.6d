advanced use - tutorial 8

 

Introducing: Advanced Line Path, Distribute Time, Line Segment Type
It is possible to allow an object (sound source or listener) to move in a pre-designated line. A line is created by setting two points at different locations and a larger line path is created by setting many points.

How?
1. Be sure Object is chosen in Object Edit
2. Choose Set Path Type as Line
3. Insert [New Point]
4. Type point position in (X,Y,Z)
5. Choose OK

You will now see a yellow line with a small yellow box, showing the line path and its finishing point. You can insert more points in this way:

How?
1. Insert [New Point] twice
2. Mark the third point in the point/time box (Blue when highlighted)
3. Mark inside the Top View window
4. Choose movement tool in Tool Bar
5. Select the little red box in the Top View window and drag the point to another position (hold left mouse button down, drag to new destination and then release)

You can move all the objects (points, sources and the listener) in this manner. If you want to set a precise position you can type the co-ordinates in the position box under Line Path

You now have a line path with straight lines, but Diesel Studio gives you the possibility of changing these straight lines into curves.

How?
1. Mark, for instance, the second point (You choose a line by marking the start point in the line)
2. Go to Line Segment Type box
3. Select Left Quarter Circle in the XY elevation (You can select between half and quarter circles) Be aware that if you have a combination with different levels e.g. elevation in the side view; the curve must look different in the top view.
