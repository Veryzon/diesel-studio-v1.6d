Introducing: Source Radiation, Orientation
In the previous tutorials the sound sources have been multidirectional, but many sound sources, for example human speech, have a higher intensity in one direction than in others. In daily life many sound sources move naturally; they are moving objects with an orientation.

How?
1. Go to File
2. Load Tutorial4.dss
3. Play

You will now hear a voice in front of you, which alternately increases and decreases in volume. Now try changing cone and orientation.

How?
1. Select 'Butterfly' in the Object Edit box
2. Go to Circle Path
3. Drag slider Object Orientation/AZ to -9
4. Go to Parameter
5. Set Inside Cone to 120
6. Set Outside Cone to 180
7. Set Outside Cone Volume to - 8
8. Select Play

You will see and hear the sound source move in the opposite direction. The volume fluctuations are now different.
