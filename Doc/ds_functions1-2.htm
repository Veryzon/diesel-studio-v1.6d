editor window - object properties

 

The object properties window is used to set all parameter data for the objects and to create and edit a path for an object. 

The object properties window is displayed as a tabbed menu with four tabs: 

Edit Object: to set the position and orientation for the selected object. 

Parameter: to set advanced parameters for the sound sources. 

Line Path: to create and set a line path. 

Circle Path: to create and set a circle path.
