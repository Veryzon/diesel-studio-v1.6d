This user's guide offers you the opportunity of obtaining optimum use from Diesel Studio.
Sections 2-4 contain a series of tutorials and sample scenes explaining, step by step, not only how to use Diesel Audio but also the various 3D-sound parameters contained in it.
Section 5 contains a detailed review of all functions. Section 6 includes various texts dealing with 3D sound and acoustics.

Product description
The Diesel Studio is a 3D audio editor used to render positional 3D audio by creating a 3 dimensional environment using a receiver/listener together with sound sources. With the Diesel Studio user interface, it is possible to simultaneously position the receiver, and a high number of sound sources in the 3D environment. Each sound source is related to a wave file and can be manipulated in the 3D environment using the mouse, or along a user defined line path or circle path. 

All parameters for achieving Positional Audio are included, i.e. volume, distance attenuation, source radiation and pitch. The playback system of Diesel Studio contains both headphone and loudspeaker playback
