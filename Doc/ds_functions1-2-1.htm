object properties - edit object

 

Edit Object
In the Edit object the object name, the wav file name (only for sound sources), and 3D model file name are displayed at the top. The selected object's position (x, y, z coordinates) and orientation (az, el, ti coordinates) can be changed in real time by tracking the slider or by typing a new value in the edit box.

The position is in meters and the range is set from -50 to 50 meters. Orientation is in degrees and the range is set from 0 to 360 degrees.
